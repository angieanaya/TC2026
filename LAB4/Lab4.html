Q & A

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    JavaScript Java
    Lenguaje interpretado basado en objetos. Lenguaje compilado orientado a objetos y de propósito general.
    Lenguaje basado en un estándar abierto para la implementación libre del mismo en cualquier sistema. Lenguaje con enfoque particular y comercial perteneciente a Oracle.
    Lenguaje levemente tipado. Lenguaje fuertemente tipado.
    Las referencias a objetos se comprueban en tiempo de ejecución. Las referencias a objetos se comprueban en tiempo de compilación.
    Es considerado un lenguaje seguro para páginas web pues no es capaz de escribir en disco duro, a excepción de las cookies.

  2. ¿Qué métodos tiene el objeto Date?
    • getFullYear()
    • getDay()
    • getMinutes()
    • getSeconds
    • toString()

  3. ¿Qué métodos tienen los arreglos?
    • .forEach: Se ejecuta una vez por cada elemento en el arreglo
    • .map: Se ejecuta una vez por cada elemento del arreglo que crea uno nuevo.
      Si no se usará el nuevo arreglo se depe optar por el método anterior.
    • .find: Retorna el valor del primer elemento del arreglo que cumpla la condición.
    • .filter: Crea un nuevo arreglo con los elementos que pasan una condición.
    • .reduce: Ejecuta una función que retorna un acumulado como valor único.
    • .some: Verifica si un elemento que cumpla la condición existe o no en el arreglo

  4. ¿Cómo se declara una variable con alcance local dentro de una función?
    function creaMensaje() {
    var mensaje = “Mensaje de prueba”;
    }
    creaMensaje();

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Si una variable se declara de manera implícita dentro de una función su alcance es global. Es mejor declarar variables como locales para evitar contaminar el objeto global.

  6. ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
    El método regex, se podría utilizar para la verificación del ingreso de datos por parte del usuario en formas presentes en la página web.
