Material design

Design language created by Google, it combines the golden rules for successful design with innovation and technology. The goal that Google has in mind is to develop a system that allows for unified user experience across all their devices and platforms.

Principles

  • Material is the metaphor.

    Material stands for the relationship between space and motion. Technology inspired by paper and ink with the intention to facilitate creativity and innovation.

  • Intuitive and natural surfaces.

    Single underlying system where surfaces and edges provide visual cues and have familiar tactile attributes.

  • Interaction through dimensionality.

    Realistic lighting to divide spaces and indicate movements without breaking rules of physics.

  • One adaptive design.

    Organizes interactions and space where different views of the system are presented according to the device used. Colors, iconography and spatial relationships should remain costant.

  • Bold, graphic and intentional content.

    It creates hierarchy, meaning, foucs and emphasizes users actions. Includes deliberate color choices, edege to edege images, typography, intentional white space.

  • Emphasize action trough color and iconography.

    Provides waypoints for the users.

  • Users inititate change.

    Reinforces the user as a primary mover.

  • Animation choreographed on main stage.

    All actions take place in a single environments to avoid objects from breaking continuity of experience.

  • Motion provides meaning

    Focuses users attention and manintains continuity with efficient and coherent transitions.

Ejercicios JavaScript

Ejercicio 1

Entrada: un número pedido con un prompt. Salida: Una tabla con los números del 1 al número dado con sus cuadrados y cubos. Utiliza document.write para producir la salida.

Ejercicio 2

Entrada: Usando un prompt se pide el resultado de la suma de 2 números generados de manera aleatoria. Salida: La página debe indicar si el resultado fue correcto o incorrecto, y el tiempo que tardó el usuario en escribir la respuesta.

Ejercicio 3

Función: contador. Parámetros: Un arreglo de números. Regresa: La cantidad de números negativos en el arreglo, la cantidad de 0's, y la cantidad de valores mayores a 0 en el arreglo.

Ejercicio 4

Función: promedios. Parámetros: Un arreglo de arreglos de números. Regresa: Un arreglo con los promedios de cada uno de los renglones de la matriz.

Ejercicio 5

Función: inverso. Parámetros: Un número. Regresa: El número con sus dígitos en orden inverso.

Ejercicio 6

Decidí hacer una lista de tareas donde cada task nuevo es un objeto que tiene de atributos el texto y un boolean completado. El objeto puede ser marcado como completado o eliminado.

Q & A

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?
    JavaScript Java
    Lenguaje interpretado basado en objetos. Lenguaje compilado orientado a objetos y de propósito general.
    Lenguaje basado en un estándar abierto para la implementación libre del mismo en cualquier sistema. Lenguaje con enfoque particular y comercial perteneciente a Oracle.
    Lenguaje levemente tipado. Lenguaje fuertemente tipado.
    Las referencias a objetos se comprueban en tiempo de ejecución. Las referencias a objetos se comprueban en tiempo de compilación.
    Es considerado un lenguaje seguro para páginas web pues no es capaz de escribir en disco duro, a excepción de las cookies.

  2. ¿Qué métodos tiene el objeto Date?
    • getFullYear()
    • getDay()
    • getMinutes()
    • getSeconds
    • toString()

  3. ¿Qué métodos tienen los arreglos?
    • .forEach: Se ejecuta una vez por cada elemento en el arreglo
    • .map: Se ejecuta una vez por cada elemento del arreglo que crea uno nuevo.
      Si no se usará el nuevo arreglo se depe optar por el método anterior.
    • .find: Retorna el valor del primer elemento del arreglo que cumpla la condición.
    • .filter: Crea un nuevo arreglo con los elementos que pasan una condición.
    • .reduce: Ejecuta una función que retorna un acumulado como valor único.
    • .some: Verifica si un elemento que cumpla la condición existe o no en el arreglo

  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    function creaMensaje() {
      var mensaje = “Esta es una variable local”;
    }
    creaMensaje();
    alert(mensaje);

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?
    Si una variable se declara de manera implícita dentro de una función su alcance es global. Es mejor declarar variables como locales para evitar contaminar el objeto global.

  6. ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?
    El método regex, se podría utilizar para la verificación del ingreso de datos por parte del usuario en formas presentes en la página web.
