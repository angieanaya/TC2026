Componentes AJAX utilizados

  • XMLHttpRequest objeto del lado del servidor usado en JavaScript para leer o mandar informacion del y hacie el servidor de forma asíncrona.
  • JavaScript para el procesamiento local
  • Document Object Model para acceder a datos dentro de la página o elementos XML a través del método getElementByTagName
  • PHP para el paso de variables, tales como la tabla de donde se pretende sacar información y será retornada al usuario via Ajax.
  • Request functions construidas a partir del user input y la cual se encarga de llamas al componente del lado del servidor para la interacción tipo Ajax.
