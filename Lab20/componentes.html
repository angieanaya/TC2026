Componentes jQuery

  • post() cuya función es de enviar y/o cargar datos de una URL externa de forma asíncroma mediante el método HTTP POST.
    Los parámetros de la función post() son: URL, paramétros, funcion de éxito.
  • done() metodo que acepta más de un argumento que puede ser ya sea una función o un array de estas y permite la ejecución de callbacks una vez que el objeto es diferido.
